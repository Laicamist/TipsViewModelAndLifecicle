## 📌 Descripción
Esta calculadora permite estimar impuestos de manera sencilla y práctica.  
El objetivo es aplicar conceptos de programación y diseño de interfaces en un proyecto funcional.
---
## ⚙️ Características
- Cálculo automático de impuestos según parámetros definidos.
- Interfaz clara y fácil de usar.
- Código modular y mantenible.
